ไม่สามารถเช็คชื่อได้

⚙️ ตั้งค่า Google Sheets
แต่ละอาจารย์ตั้งค่าของตัวเองได้ที่นี่ ข้อมูลจะบันทึกลง Google Sheet ของแต่ละท่านแยกกัน
1
เปิด Google Sheet "ต้นฉบับเช็คชื่อ" → Extensions → Apps Script
2
ลบโค้ดเดิมทั้งหมด วางโค้ดด้านล่าง → กด Ctrl+S
3
Deploy → New deployment → Web app
Execute as: Me | Who has access: Anyone
4
คัดลอก Web App URL มาวางด้านล่าง แล้วกดบันทึก
URL จากขั้นตอน Deploy ขึ้นต้นด้วย https://script.google.com/macros/s/
อยู่ระหว่าง /d/ และ /edit ใน URL ของ Sheet
ชื่อ tab ที่ต้องการบันทึกผลเช็คชื่อ
📋 โค้ด Apps Script
คลิกที่โค้ดเพื่อคัดลอก
📤 วิธีแชร์ให้อาจารย์ท่านอื่น
1
อาจารย์แต่ละท่านเปิดไฟล์ index.html นี้ใน Chrome
2
ไปที่ ⚙️ ตั้งค่า แล้วใส่ Script URL และ Sheet ID ของตัวเอง
3
กด บันทึกการตั้งค่า — ระบบจะจำค่าไว้ในเบราว์เซอร์ของแต่ละคน
4
อัปโหลดไฟล์นี้ขึ้น GitHub Pages เพื่อให้ QR สแกนได้จากมือถือนิสิต
💡 แนะนำ: อัปโหลดไฟล์ขึ้น GitHub Pages (ฟรี) แล้วได้ URL แบบ https://username.github.io/attendance/ นิสิตสแกน QR แล้วเปิดหน้าเว็บเช็คชื่อได้ทันที
